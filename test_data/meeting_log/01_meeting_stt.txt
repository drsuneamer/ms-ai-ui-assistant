음 네 일단 지금 메인 페이지 버튼 배치가 좀 애매하다는 의견이 있어서요. 그러니까, 사용자들이 처음 들어왔을 때 어디를 눌러야 할지 헷갈려 한다는 거죠. 그래서, 어... 그 버튼을 오른쪽 상단에서 좀 더 중앙 쪽으로 옮기는 게 어떨까 싶거든요.
근데 그렇게 하면 로그인 버튼이랑 겹치지 않나요? 그 부분도 좀 고려해야 할 것 같아서요. 그니까 지금은 로그인 버튼이 오른쪽 상단에 고정되어 있는데, 그거 옮기면 또 사용자들이 익숙하지 않아서 불편해질 수 있으니까. 음, 잠깐만요. 우리가 지난번에 Figma에서 공유했던 그 버전 있잖아요. 그거 보면 메인 CTA가 가운데 아래쪽에 있었던 것 같은데, 그거 다시 참고해서 위치 조정하는 게 나을 것 같기도 하고.
아 맞다. 그리고 색상 말인데요. 현재 메인 컬러가 파란색 계열인데 약간 너무 차가운 느낌이 있다는 피드백도 있었어요. 혹시 약간 따뜻한 색 계열로 바꿔볼 수 있을까요? 예를 들어 뭐... 연한 오렌지 톤이라든지. 사용자들이 좀 더 편하게 느낄 수 있는 색상이면 좋을 것 같아서요.
그런데 또 너무 많은 색을 쓰면 가독성 문제가 생기니까 조심해야 될 것 같고요. 그리고 버튼 색이랑 텍스트 색이랑 대비가 충분한지도 꼭 체크해야 하고요. 특히 모바일 환경에서는 더더욱. 지금 저해상도 기준에서 테스트했을 때 약간 흐릿하게 보이는 경우가 있더라고요.
아 그리고 메뉴 구조 얘기 나왔던 거 혹시 기억나세요? 지금은 햄버거 메뉴 안에 거의 모든 기능이 다 들어가 있어서... 처음 들어오는 사람은 뭐가 어디 있는지 잘 몰라요. 우리가 기능적으로는 많이 담았는데 정작 UX적으로는 직관성이 떨어진다는 말도 있었고. 음, 그니까 메인 메뉴에서 자주 쓰는 기능은 밖으로 꺼내놓는 게 나을 것 같아요. 예를 들면 '설정'이나 '내 정보' 같은 건 고정 노출하는 쪽으로요.
네, 저도 그 의견 동의하고요. 특히 사용자 피드백 중에 "탭 이동이 너무 많다"는 의견이 좀 많았어요. 그래서 하나의 흐름 안에서 자연스럽게 다음 화면으로 넘어갈 수 있도록 구조 자체를 단순화할 필요가 있어 보여요.
그리고 어제 QA 팀에서 얘기한 거 있었잖아요. 일부 버튼이 터치 영역이 너무 작아서 오작동 빈도가 높다고. 특히 태블릿에서. 그래서 버튼 최소 크기 가이드라인 다시 확인하고, 지금 구현된 UI랑 비교해봐야 할 것 같아요. 안드로이드 기준으로는 최소 48dp였던 것 같은데요.
음 그리고 모달창 쪽 피드백도 있었는데, 모달이 떴을 때 배경이 너무 어두워서 내용이 잘 안 보인다는 얘기가 있었어요. 그거는 배경 블러 처리라든가 투명도 조절로 해결할 수 있을 것 같긴 한데, 디자이너랑 다시 얘기해봐야 될 것 같고요.
그리고 접근성 관련해서 텍스트에 alt 태그 빠져 있는 거 발견됐다고 했죠? 그 부분은 빠르게 수정해야 할 것 같아요. 특히 공공기관 납품 일정 맞추려면 그런 부분에서 점수 깎이면 안 되니까.
아 그리고 사용자 흐름 테스트 지금 A안이랑 B안 두 개로 나눠서 하고 있잖아요. 근데 A안 쪽에서 이탈률이 더 높게 나오고 있어요. 클릭률은 비슷한데 진행률에서 차이가 나고 있어서, 그 이유를 좀 더 분석해 봐야 할 것 같아요. 어쩌면 화면 간 연결이 좀 덜 매끄럽거나, 불필요한 요소가 있을 수도 있고요.
맞아요. 특히 그 중간에 나오는 광고 배너가 좀 크고 거슬린다는 의견도 있었어요. 이거는 제품 쪽이랑 다시 협의해야 할 것 같고요. 광고 위치나 노출 빈도를 조절할 수 있을지 확인해보는 게 좋겠어요.
그리고 모바일에서 키보드 올릴 때 버튼 가려지는 문제는 아직 수정 안 됐죠? 그건 기술팀이랑 우선순위 조정해서 바로 처리해야 할 것 같고요. 테스트할 때 환경별로 다 확인해봤으면 좋겠어요. 안드로이드, 아이폰, 그리고 태블릿까지.
잠깐, 그러면 이거 정리해서 Notion에 올릴까요? 각 항목별로 태스크로 나눠서 담당자 지정하면 바로 작업 들어갈 수 있을 것 같은데요. 음... 그럼 제가 화면 구조 관련해서는 먼저 정리해볼게요. 색상은 디자이너 쪽에서 의견 한 번 더 받아보는 게 좋을 것 같고요.
좋아요, 그럼 오늘 회의 요약은 제가 먼저 적어보고 공유드릴게요. 부족한 부분은 코멘트로 남겨주시면 반영하겠습니다.
